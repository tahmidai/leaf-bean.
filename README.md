# Leaf & Bean - Tea & Coffee Journal

A modern, responsive 4-page website with dark café aesthetic, minimal design, and magazine-style layout.

## Live Demo
https://[your-username].github.io/leaf-bean-website/

## Features
- 4 responsive pages: Home, Teas, Coffees, Brew Guide
- Dark café aesthetic with warm accent colors
- Card-style sections with hover animations
- Sticky navigation with mobile menu
- Fully responsive design

## Pages
1. **Home** - Introduction, featured items, mission statement
2. **Teas** - Types of tea (Green, Black, Oolong, Herbal/Chai) with descriptions
3. **Coffees** - Types of coffee (Arabica, Robusta, Espresso, Brew methods)
4. **Brew Guide** - Brewing tips, common mistakes, pro tips

## Technologies
- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- JavaScript (for mobile navigation)
- Font Awesome icons
- Google Fonts (Playfair Display, Inter)
- Unsplash images

## How to Run Locally
1. Clone repository: `git clone https://github.com/[your-username]/leaf-bean-website.git`
2. Open `index.html` in any modern web browser
